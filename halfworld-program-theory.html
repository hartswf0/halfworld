00The Name

HALFWORLD. Halftone — the shared substrate: every artifact renders ink dots on paper. HALFTRACK — the main artifact for characters, whose embedding API is the engine's spine. World — what everything builds toward. And the diagnosis itself: each of the nine artifacts is half a world. Characters without places. Places without characters. Scores without stages. Minds without bodies. The engine is the weld.

Definition

HALFWORLD is a paper-and-ink performance engine in which character ⊂ scene ⊂ world, every layer is plain JSON, and the same small verb set — pose, walk, face, say, place, remember — is spoken equally by a hand on the screen, a game loop, a wall-clock score, and an AI mind emitting bounded operation packets.

01Diagnosis

Nine artifacts, four families, one aesthetic, and three fractures.

Fracture one — three character systems that cannot meet. HALFTRACK has the richest figure rig (spec → metrics → 40 poses, IK arms, silhouette hands, four halftone styles). Atlantis grew its own identity-forward cast system (pronouns, wardrobe, ten skin tones, access markers, an articulated dot rig) that even names its drives after HALFTRACK's — but shares no schema with it. SYNTHCINE's ICARO rig has the best faces (visemes, affect, head-carriage springs) bound to five fixed characters and one canonical donor head. A HERO made in HALFTRACK cannot walk into Atlantis, and NAT cannot leave her call tile.

Fracture two — three clocks that cannot agree. Whispers runs live and reactive (ripples, no timeline). The Call runs scored and scrub-deterministic (every frame derived from the playhead). Virentis runs turn-based (one packet per utterance). The comedy engine layers a beat graph over a score and proves the clocks can nest — but only inside one file.

Fracture three — no shared memory or bus. HALFTRACK and Worldsmith persist nothing. Atlantis exports project JSON; Virentis exports world JSON; the Call persists mocap takes in localStorage — five stores, no common schema, and not one BroadcastChannel or postMessage between any two artifacts.

02Theory of Change

The artifacts already contain their own solution. Each one, independently, arrived at the same move: reduce the thing to a small JSON contract and let anything speak it. HALFTRACK's window.HALFTRACK. Atlantis's project snapshot. The Call's OTIO line manifest. Virentis's world packet. The comedy engine's beat graph. These are not nine programs; they are five contracts wearing nine costumes.

The Wager

IF the five contracts are lifted out of their host files and made canonical — one CAST, one STAGE, one SCORE, one MIND, one MEMORY — THEN any façade becomes a restageable scene, any character can walk into any world, any clock can drive any stage, and any mind can direct any cast — BECAUSE every artifact already proves its own contract works; what has never existed is the weld between them.

The unit of progress is therefore not "a bigger app." It is one contract at a time, demonstrated by one seam at a time — the opposite of half-track's everything-at-once attempt, which the diagnosis says falls short precisely because it swallowed the world instead of contracting with it.

03The Five Contracts
// 1 · CAST — who exists. donor: HALFTRACK spec + Atlantis identity + ICARO face
{ id, identity:{ name, pronouns, culture, access },
  look:{ hair, skin, garment, build, height, glasses, tone },   // HALFTRACK spec dials
  face:{ style, skinInk, hairTone },                            // ICARO casting dials
  rig:"halfworld-pose-1" }                                      // one channel set, union of
                                                                // mkSignals / mkPose / performance.*

// 2 · STAGE — what exists where. donor: Atlantis asset/instance/world + Worldsmith generator
{ assets:{ id:{ kind, points|grid|cmds, childWorldId } },       // dots are the geometry
  instances:[{ assetId, x, z, y, scale, rot, layer, anim }],
  cast:[{ castId, x, z, pose, facing }],                        // people are placements too
  layers:[ ground..interface ] }                                // Atlantis's 12, executable

// 3 · SCORE — when things happen. donor: SYNTHCINE LINES (the MONTE transport pattern)
{ fps:24, dur, rows:[{ id, c, s, d, t, m:"cue@word; cue=param", wpm }] }
                                                                // scrub-deterministic:
                                                                // state = f(playhead), never queued

// 4 · MIND — who decides. donor: Virentis packet + whispers ripples + comedy BEATS
{ reply, changes:[ op ≤ 8, validated, clamped ],               // ops target STAGE + CAST + SCORE
  memory:[], suggestions:[] }
beats:[{ id, pre(world), lines:[{ c, text, phrase, reactions }] }]

// 5 · MEMORY — what persists. donor: Atlantis project JSON + ledger
{ schema:"halfworld/v1", cast, stages, scores, beats, worlds,
  operations:[ append-only ledger ], field:{ virentis tiles } }

Two laws bind the contracts. Law of the dot: all geometry bottoms out in dots on paper — Worldsmith grids, Atlantis point clouds, HALFTRACK tone screens, ICARO halftone fields are one substrate at four densities, so any renderer can draw any contract. Law of the packet: minds never touch pixels. An AI director holds exactly the powers a finger has — it emits ops and cues against the contracts, bounded and validated, the way Virentis already clamps its planet to eight changes a turn.

05Seams to Weld
  • Cast → Studio. Adapter from HALFTRACK spec to Atlantis's character record; HALFTRACK's figure renderer drawn at an Atlantis instance transform. The proof: HERO stands in a recursive world. Atlantis's performance.* fields already accept external drives — this seam is the cheapest and unlocks the most.
  • Cast → Façade. ICARO casting is only hair + glasses + tone over one donor head; extend cast() to read a CAST record so Scene 06 can be performed by any characters, not five fixed ones.
  • Score → everything. Whispers' agent outputs and the comedy engine's beat lines both compile down to SCORE rows; a recorded improvisation becomes a scrubbing score. Live and scored stop being different apps and become record/playback of one format.
  • Mind → everything. One packet schema whose ops target STAGE (place, terrain), CAST (spawn, act), and SCORE (insert row). Virentis's validator is the reference implementation.
  • Memory → everything. One halfworld/v1 snapshot; every artifact gains load/save of the sections it understands. Atlantis's playable-HTML export becomes the engine's publishing move.
  • The bus. Nothing talks today. A single BroadcastChannel (halfworld-bus) carrying ops and score rows lets the existing files cooperate before any of them merges — the MONTE director-console pattern, generalized.
  • The key. Three artifacts call model APIs directly from the page (and one ships no auth at all). One shared model-endpoint setting — local-first like Virentis, remote when keyed — replaces three ad-hoc implementations.
06Build Path
  1. Contracts on paper Write halfworld/v1 as a single spec file with the five schemas and the validator. No app changes — just the truth, versioned.
  2. The cast weld HALFTRACK character appears in Atlantis via the adapter, placed, scaled, layered, saved into project JSON. First time a character crosses a file boundary.
  3. The scored room Rebuild the BeFliX room as the first native HALFWORLD stage: room geometry authored in Worldsmith/Atlantis tools (not literal code), cast drawn from CAST records, LINES driving them, beats above, packets editing the room live. This is the artifact the façades were reaching for.
  4. The field above Virentis becomes the atlas: each tile-feature can open onto a stage; expeditions link scenes into the worldtext. Character ⊂ scene ⊂ world, navigable end to end.
07Principles
  • Dots are the substrate. Ink on paper at every scale — a mouth, a person, a house, a planet. No artifact may introduce geometry that cannot be expressed as dots.
  • Contracts, not merges. The failure mode is the everything-app. Artifacts stay small and single-minded; they cooperate by speaking the five contracts over the bus and the snapshot.
  • Scrub-determinism. Anything performed must be derivable from a playhead position. Live play is recording; playback is the score. Never queued timers.
  • Minds are bounded. A model gets the same verbs as a finger, validated and clamped. The more powerful the director, the more important the contract.
  • Everyone is castable. Identity is data — pronouns, culture, access, wardrobe — and every stage must accept every CAST record, not a fixed roster.
  • Start with nothing. Any word must produce something. Generation is a promise, not a menu.
